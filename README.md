# trust' - Карта доступности города

Frontend приложение для карты доступности города для людей с ограниченными возможностями передвижения и пользователей СИМ.

## Технологии

- React 18
- React Router DOM
- Leaflet (интерактивная карта)
- Vite (сборщик)
- CSS3 (адаптивная верстка)

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

Запустите приложение в режиме разработки:
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Сборка

Для создания production сборки:
```bash
npm run build
```

## Структура проекта

```
2jis-frontend/
├── src/
│   ├── components/      # Переиспользуемые компоненты
│   │   ├── Header.jsx   # Шапка сайта
│   │   └── Map.jsx      # Компонент карты
```

## Функциональность

- ✅ Интерактивная карта с маркерами мест
- ✅ Страница регистрации и входа
- ✅ Страница "О нас"
- ✅ Детальная информация о местах с рейтингом и отзывами
- ✅ Страница профиля пользователя
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Навигация между страницами

## API

Приложение готово к интеграции с backend API. Настройте прокси в `vite.config.js` для работы с вашим API сервером.

## Примечания

- Для работы карты требуется интернет-соединение (используется OpenStreetMap)
- Данные о местах в настоящее время используются моковые
- Аутентификация еще не реализована (заглушка)
